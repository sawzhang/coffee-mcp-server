Available Tools

search_nearby_stores
Search nearby Starbucks stores by city, keyword, or coordinates
"Find Starbucks near Lujiazui, Shanghai"
get_store_detail
Store details: address, hours, services, contact
"What time does this store close?"
get_menu
Browse menu by category: espresso, seasonal, tea, food
"What new drinks are available this season?"
get_product_detail
Product details: sizes, prices, calories, customizations
"Can I get oat milk in a Flat White?"
check_store_inventory
Check product availability at a specific store
"Is the Sakura Latte available today?"
get_promotions
Current promotions: BOGO, discounts, member offers
"Any buy-one-get-one deals right now?"

Quick Start

1

Clone & Install

Requires uv and Python 3.13+

$ git clone https://github.com/sawzhang/starbucks-mcp-server.git $ cd starbucks-mcp-server $ uv sync
2

Run the Demo

Execute all 6 tools in sequence

$ uv run sbux demo # Step 1: Search Shanghai stores # Step 2: Get store detail # Step 3: Browse seasonal menu # Step 4: Product detail (Latte) # Step 5: Check inventory # Step 6: Current promotions
3

Connect to Claude Code / Cursor

Add to your MCP config file

// Claude Code: ~/.claude/claude_code_settings.json // Cursor: .cursor/mcp.json { "mcpServers": { "starbucks": { "command": "uv", "args": ["--directory", "/path/to/starbucks-mcp-server", "run", "starbucks-mcp"] } } }

Real-World Scenario

User:
"I have a meeting near Lujiazui, pre-order a grande oat latte with less ice at the nearest Starbucks"

AI Agent executes:

Step 1: search_nearby_stores(city="Shanghai", keyword="Lujiazui") → 3 stores found
Step 2: get_store_detail(store_id="SH-LJZ-001") → Open, supports pickup
Step 3: check_store_inventory(store_id="SH-LJZ-001", product_id="ESP-002") → In stock
Step 4: get_product_detail(name="Latte") → Grande ¥37, oat milk +¥5
"Bund Raffles City store (8 min walk from Lujiazui), grande oat latte with less ice, order #A042, ready at 14:30."
